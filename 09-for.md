# For

Muy parecido al switch se ejecuta hasta que se cumpla una condicion especifica.

Se ejecutara mientras a sea <= que b.

```java
public class For {
  public static void main(String[] args) {
    int a;
    int b = 12;

    for (a = 1; a <= b; a++) {
      System.out.println("A sigue siendo menor a b, valor a: " + a);
    }
  }
}
```
