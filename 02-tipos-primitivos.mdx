# Tipos Primitivos

Una variables es un contendor en el que le almacenas un dato, en otras palabras es un espacio en memoria.

| Tipo      | Rango / Valor aproximado                               | Tamaño  | Ejemplo                  |
| --------- | ------------------------------------------------------ | ------- | ------------------------ |
| `byte`    | -128 a 127                                             | 8 bits  | `byte b = 100;`          |
| `short`   | -32,768 a 32,767                                       | 16 bits | `short s = 32000;`       |
| `int`     | -2,147,483,648 a 2,147,483,647                         | 32 bits | `int i = 100000;`        |
| `long`    | -9,223,372,036,854,775,808 a 9,223,372,036,854,775,807 | 64 bits | `long l = 10000000000L;` |
| `float`   | ±3.4×10³⁸ (precisión \~6-7 decimales)                  | 32 bits | `float f = 3.14f;`       |
| `double`  | ±1.7×10³⁰⁸ (precisión \~15-16 decimales)               | 64 bits | `double d = 3.14159;`    |
| `char`    | Caracter Unicode de 16 bits (`'\u0000'` a `'\uffff'`)  | 16 bits | `char c = 'A';`          |
| `boolean` | `true` o `false`                                       | 1 bit\* | `boolean b = true;`      |
